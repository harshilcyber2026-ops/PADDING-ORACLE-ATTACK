A vulnerable server is running an AES-128-CBC decryption endpoint. After decrypting your submitted ciphertext, it only tells you one thing — whether PKCS#7 padding is valid or invalid. No plaintext is ever returned.


This is a classic Padding Oracle scenario. Using the oracle's yes/no response, a skilled attacker can recover the full plaintext byte-by-byte without the key. The encrypted flag is waiting to be extracted.

Attack Guide

  

    

      
01

      
Understand CBC Decryption

      
In AES-CBC, plaintext block N = Decrypt(C_N) XOR C_{N-1}. The IV is C_{-1}. You fully control the IV bytes you send to the oracle.

    

    

      
02

      
Flip IV Bytes

      
Modify the last byte of the IV (0–255). When the oracle returns VALID, \x01 padding was achieved. Solve: P[15] = 0x01 XOR guess XOR IV[15]

    

    

      
03

      
Recover All 16 Bytes

      
For each padding target (0x01→0x10), recover one more byte. Work right-to-left across all 16 bytes of each block.

    

    

      
04

      
Automate with Python

      
Manual attacks need up to 256×16 queries. Write a script using the template below to automate oracle calls and recover the full plaintext.

# Padding Oracle Attack — FlagVault CTF
import requests, base64

ORACLE_URL = "https://harshilcyber2026-ops.github.io/RSA-Weak-Key/"
CT_B64     = "obLDpOX2BxgpOktcbX6PkKGyw6Tl9gcYKTpLXG1+j5CxsJ+Of45/jn+Pj4+Qj5GPkA=="
CT_BYTES   = base64.b64decode(CT_B64)

def oracle(ct_bytes):
    r = requests.post(ORACLE_URL + "oracle",
        json={"ct": base64.b64encode(ct_bytes).decode()})
    return r.json().get("valid") == True

def decrypt_block(prev_block, curr_block):
    intermediate = bytearray(16)
    plaintext    = bytearray(16)
    for pad_byte in range(1, 17):
        crafted_suffix = bytes([intermediate[16-i] ^ pad_byte
                         for i in range(1, pad_byte)])
        for guess in range(256):
            crafted = bytearray(16-pad_byte) + bytes([guess]) + crafted_suffix
            if oracle(bytes(crafted) + bytes(curr_block)):
                intermediate[16-pad_byte] = guess ^ pad_byte
                plaintext[16-pad_byte]    = intermediate[16-pad_byte] ^ prev_block[16-pad_byte]
                break
    return bytes(plaintext)

blocks    = [CT_BYTES[i:i+16] for i in range(0, len(CT_BYTES), 16)]
plaintext = b""
for i in range(1, len(blocks)):
    plaintext += decrypt_block(blocks[i-1], blocks[i])
print("[+] Flag:", plaintext)
